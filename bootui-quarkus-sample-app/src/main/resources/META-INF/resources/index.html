Quarkus demo

Welcome to the BootUI sample app

This app demonstrates a small Quarkus service with public endpoints, secured admin routes, scheduled work, Hibernate ORM data, and enough runtime state for BootUI to inspect.

Open BootUI

Sample action lab

Run a few safe requests to generate data for BootUI panels. Then open the linked panels to see how BootUI explains the same app from different angles.

Calls GET /api/sample/products to exercise Hibernate ORM, Quarkus Cache, and HTTP Exchanges.

Inspect: Cache, SQL Trace, HTTP Exchanges

Calls GET /api/sample/product-search to run a live, uncached SQL SELECT.

Inspect: SQL Trace, Database Connection Pools

Calls GET /api/sample/hello so BootUI can show configuration-backed behavior.

Inspect: Configuration, Cache, HTTP Probe

Calls GET /api/hello as a simple public endpoint for mappings and probes.

Inspect: Mappings, HTTP Probe, HTTP Exchanges

Calls GET /api/secure with admin/admin credentials.

Inspect: Security, Security Logs

Calls GET /api/secure/products with admin/admin to run a secured, SQL-backed request.

Inspect: Live Activity, Security, SQL Trace

Calls the same secured endpoint with developer/developer to produce a forbidden response.

Inspect: Security, Security Logs, Pentesting

Calls GET /api/sample/boom, which throws on purpose to record a server-side failure.

Inspect: Exceptions, Log Tail, Traces

Calls GET /api/sample/metrics-burst to record a custom counter and timer several times.

Inspect: Metrics

Calls GET /api/sample/allocate to briefly allocate a bounded heap buffer.

Inspect: Live Memory, Heap Dump, JVM Tuning

Calls GET /api/sample/slow, which sleeps briefly so a request thread stays busy.

Inspect: Threads, Metrics, HTTP Exchanges

Calls GET /api/sample/pool-stress to run several concurrent read-only queries.

Inspect: Database Connection Pools, SQL Trace

Calls GET /api/sample/chained, which nests spans across a catalog read and a live query.

Inspect: Traces, Diagnostics, HTTP Exchanges, SQL Trace

Calls GET /api/sample/send-email to send a demo email through the app's mailer.

Inspect: Email

Calls GET /api/secure with bad credentials to produce an unauthorized response.

Inspect: Security, Security Logs, Pentesting
