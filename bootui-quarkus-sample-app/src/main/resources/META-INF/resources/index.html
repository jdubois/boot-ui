BootUI Quarkus Sample App

A reference Quarkus application for the BootUI developer console.

Open the BootUI console →

Exercise the demo endpoints to give the panels something to show:

  • GET /api/sample/hello: cached greeting
  • GET /api/sample/products: cached catalog query (Quarkus Cache)
  • GET /api/sample/metrics-burst: custom Micrometer meters (Metrics)
  • GET /api/sample/allocate?mb=32: heap bump (Live Memory)
  • GET /api/sample/slow?ms=1500: busy thread (Threads)
  • GET /api/sample/pool-stress: concurrent queries (DB Connection Pools)
  • GET /api/sample/chained: nested spans (Traces)
  • GET /api/sample/boom: thrown exception (Exceptions)
  • GET /api/architecture/some-errors: advisor findings
  • POST /api/chat: LangChain4j/Ollama (AI Usage)

Secured endpoints (/admin, /api/secure) require the admin/admin account.
